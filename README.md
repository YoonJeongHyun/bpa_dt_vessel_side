### 2. 선박 최적항로 도출
### 3. 최적 선석 계획 도출

## Environment
- recommmended python version : Python 3.7
- 세부 패키지는 requirements.txt 참조 (설치방법 : https://engineer-mole.tistory.com/258)


## Run guide

`python main.py pnit`

- 외부 테스트   
`python main.py`
